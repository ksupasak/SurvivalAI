07 — Bluetooth and Camera Permissions

The App requires certain device permissions to function:

  • Camera: Used for Morse code light signal detection (Read mode). Camera feed is processed entirely on-device and is never transmitted or stored.
  • Flashlight (Torch): Used to transmit Morse code signals.
  • Bluetooth (BLE): Used in Seeking Mode to broadcast a rescue beacon. No data is stored in BLE advertisements beyond what you explicitly configure in your profile.
  • Location: Used optionally to include GPS coordinates in the BLE beacon for rescue teams. Location data is not uploaded to any server.
  • Microphone: Used for voice-to-text input in the AI chat. Audio is processed on-device or through your chosen AI provider's API and is not stored by SurvivalAI.
