Section 02

Description of Service

SurvivalAI is an offline-first mobile application designed to support individuals in emergency, disaster, and survival situations. The App runs on iOS 15+ and Android 10+ and provides the following core modules:

  • Offline AI Chat — On-device large language model (LLM via llama.rn) with survival knowledge covering first aid, shelter, water, navigation, and more. Optional online mode via a user-supplied API key.
  • Supply Calculator — Calculates water, food, first-aid, hygiene, shelter, and tool quantities based on group size, duration, climate, and activity level.
  • Morse Code SOS — Transmits Morse code via device flashlight; receives and decodes Morse code light signals via the rear camera with an adjustable sensitivity targeting reticle.
  • Seeking Mode — Broadcasts a Bluetooth Low Energy (BLE) rescue beacon containing optional emergency profile data and GPS coordinates. Includes an audio beacon and battery-preservation screen-dimming.
  • Emergency Profile — Locally stores personal medical data (name, blood type, allergies, medications, emergency contacts) for BLE and Morse transmission.
  • Document Library — Downloads official government survival manuals (FEMA, Red Cross, FM 21-76, WHO) for fully offline reference.

The Service is provided free of charge, with no subscription or in-app purchases. The App is not a medical device, a licensed emergency communication system, or a substitute for professional rescue services.
