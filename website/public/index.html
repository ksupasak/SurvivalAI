// Personal Profile

Your Lifeline Data

Store critical medical and personal information that rescuers can access via BLE beacon or Morse code.

Information Rescuers Need

In an emergency, first responders need quick access to your medical data. SurvivalAI stores your profile locally and broadcasts it via BLE when Seeking Mode is active.

  • Blood type and medical conditions
  • Current medications and allergies
  • Emergency contact information
  • Special needs and accessibility requirements
  • Transmit via Morse code flashlight
  • Broadcast via BLE to rescue teams
  • All data stored locally on device only
